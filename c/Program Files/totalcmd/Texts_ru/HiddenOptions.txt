Open File shell for TC, v1.8b

Опции, не вошедшие по каким-либо причинам в окно настроек программы.
Для использования опций вам нужно открыть файл настроек OpenFileTC.ini
и вручную добавить в соответствующую секцию требуемое поле.

Свежие поля находятся сверху

----- 

Начиная с версии 1.8с (1.8.5.38), OpenFileTC поддерживает экспорт выделенных
файлов в текстовый файл и передачу полного пути до этого текстового файла
редактору. Для этого редактору напротив требуемого расширения необходимо 
задать командную строку следующего вида: %filelist=<здесь полный путь до
создаваемого файла>%. Но сам параметр можно укоротить - задать просто
%filename% вместо длинной записи выше. В этом случае OFTC создаст 
файл-список во временной системной папке с именем по умолчанию.
Параметр отвечает за короткое название данного файла-списка по умолчанию, 
сам файл будет создаваться во временной папке.
[Main]
DefaultListFileName=oftc_list.txt

-----

Параметр отвечает за вызов вопроса пользователю - хочет ли он проверить 
настройки своих приложений (в OFTC) по базе Applications.ini. В принципе, 
программа сама спрашивает 1 раз при первом старте новой версии, и в 
дальнейшем больше не спрашивает. Если выставить данный флаг в ноль, то 
программа снова задаст вопрос о перепроверке
По умолчанию отключен, становится равным единице после реакции пользователя
[Main]
AlreadyCheckedRunTypesByAppBase=0

-----

Параметр отвечает за подсчёт количества вызовов программ. Если
включить эту опцию, то программа перестанет считать количество 
вызовов и не будет сортировать по этому параметру
По умолчанию отключен
[Main]
DisableCallsCount=0

-----

Поле отвечает за фиксацию всех компонентов на окнах при изменении
размеров окна. Влияет на вспомогательные окна, на главное окно не 
оказывает никакого влияния.Может помочь вам, если, к примеру, у вас 
нестандартный шрифт и кнопки уезжают за край окон.
По умолчанию отключен.
[Main]
FixWindows=0

-----

Если вы включали проверку на мышь для принудительного открытия 
главного окна, то эта опция позволяет установить чувствительность 
при проверке позиции мыши. Параметр указывает, в каких пределах 
от соответствующего угла (в точках) будет находиться курсор мыши.
По умолчанию оба параметра равны 18 (т.е. проверяется угол 18х18).
[ForceApp]
MouseDX=18
MouseDY=18

-----

Параметр отвечает за кол-во отображаемых расширений в подсказке на 
одну строку. На первой строке будет на 3 расширения меньше, чем 
этот параметр. По умолчанию 7.
[Main]
ExtsCountPerString=7

-----

Параметр отвечает за получение списка выделенных файлов. По умолчанию,
программа старается при получении списка выделенных файлов не 
использовать буфер обмена. Но в особых случаях без буфера обмена не
обойтись. Чтобы программа могла обращаться к буферу обмена и получать
список файлов через него, выставите это значение равным единице.
Пол умолчанию ноль. 
Внимание! Если эта опция будет равна единице, то после запуска 
программы возможны потери буфера обмена!
[Main]
CanUseClipboard=0
