Tastaturbelegung von Total Commander
====================================

Taste        Wirkung

F1           Hilfe aufrufen
F2           Quellverzeichnis neu einlesen
F3           Datei(en) anzeigen
F4           Datei editieren
F5           Datei(en) kopieren
F6           Datei(en) umbenennen/verschieben
F7           Verzeichnis erstellen
F8/ENTF      Datei(en) löschen in Papierkorb/direkt - je nach Konfiguration
F9           Menü oberhalb Quellverzeichnis aktivieren (links bzw. rechts)
F10          Linkes Menü aktivieren bzw. Menü deaktivieren
ALT+F1       linkes Laufwerk wählen
ALT+F2       rechtes Laufwerk wählen
ALT+F3       umschalten zwischen externem/internem Dateibetrachter
ALT+UMSCH+F3 Starte Lister mit interner Anzeige ohne Plugins oder Multimedia
ALT+F4       Beenden|Minimieren (mit Option MinimizeOnClose, siehe Hilfe)
ALT+F5       Packen
ALT+UMSCH+F5 Packen und anschließend löschen
ALT+F6       Mehrere Dateien aus Archiv unter Cursor entpacken
ALT+F7       Suchen
ALT+UMSCH+F7 Suchen in separatem Prozess
ALT+F8       Klappt History-Liste der Kommandozeile auf
ALT+F9       Wie ALT+F6 (weil ALT+F6 unter Windows 95 nicht geht!)
ALT+UMSCH+F9 Archive auf Integrität testen
ALT+F10      Öffnet ein Fenster mit dem Verzeichnisbaum
ALT+F11      Linke Verzeichnisleiste wählen (Brotkrumenleiste)
ALT+F12      Rechte Verzeichnisleiste wählen (Brotkrumenleiste)
ALT+UMSCH+F11
             Buttonbar fokussieren (zum Benutzen mit der Tastatur)
ALT+links/
    rechts   Vorheriges Verzeichnis/nächstes Verzeichnis der bereits besuchten
ALT+ab	     Zeige Liste der bisher besuchten Verzeichnisse an
ALT+Umsch+ab Zeige Liste der bisher besuchten Verzeichnisse ohne Ausdünnen an
UMSCH+F1     Menü "Benutzerdefinierte Spaltenansicht"
UMSCH+F2     Verzeichnisse vergleichen
UMSCH+F3     Nur Datei unter Cursor anzeigen, wenn mehrere markiert sind
UMSCH+F4     Neue Textdatei anlegen und in Editor laden
UMSCH+F5     Datei(en) im gleichen Verzeichnis kopieren (mit umbenennen)
UMSCH+STRG+F5
             Verknüpfung(en) der Datei(en) im Zielverzeichnis erzeugen
UMSCH+F6     Datei(en) im gleichen Verzeichnis umbenennen
UMSCH+F7     Verzeichnis im anderen Dateifenster erstellen, mit Namen unter Cursor
UMSCH+F8/ENTF
             Datei(en) löschen, direkt/in Papierkorb - je nach Konfiguration
Umsch+F10    Lokales Menü anzeigen
UMSCH+ESC    Total Commander wird zum Symbol verkleinert

NUM +        Markierung erweitern (konfigurierbar ob nur Dateien oder auch Verzeichnisse)
NUM -        Markierung verkleinern (immer von Dateien und Verzeichnissen)
NUM *        Markierung umkehren (auch mit Umsch)
NUM /        Markierung wiederherstellen
STRG+NUM +   alles markieren
STRG+UMSCH+NUM +
             Wie STRG+NUM +, aber Dateien+Verzeichnisse, wenn STRG+NUM + nur alle Dateien markiert
STRG+NUM -   nichts markieren
STRG+UMSCH+NUM -
             Markierung nur von Dateien entfernen (nicht von Verzeichnissen)
UMSCH+NUM +  Wie NUM +, aber Dateien+Verzeichnisse, falls NUM + nur die Markierung von Dateien umkehrt
UMSCH+NUM -  Entfernt Markierung nur von Dateien (immer!)
UMSCH+NUM *  Wie NUM *, aber Dateien+Verzeichnisse, falls NUM * nur die Markierung von Dateien umkehrt
ALT+NUM +    Dateien mit selber Erweiterung markieren
ALT+NUM -    Markierung von Dateien mit selber Erweiterung entfernen
BACKSPACE/
STRG+BildAuf ins nächsthöhere Verzeichnis wechseln (entspricht cd ..)
STRG+BildAb  Verzeichnis/Archiv öffnen (auch .EXE)
STRG+<       Ins Hauptverzeichnis (Wurzelverzeichnis) wechseln
STRG+links/
     rechts  Öffnet Verzeichnis/Archiv unter dem Cursor und zeigt es im
             Zielfenster an. Falls sich der Cursor nicht auf einem
             Verzeichnis/Archiv befindet, oder im Zielfenster, wird 
             stattdessen das aktuelle Verzeichnis angezeigt.
STRG+F1      Dateiansicht kurz
STRG+Umsch+F1
             Miniaturansicht (Vorschaubilder)
STRG+F2      Dateiansicht lang
STRG+Umsch+F2
             Kommentaransicht (Kommentare mit Strg+Z anlegen)
STRG+F3      Sortieren nach Name
STRG+F4      Sortieren nach Erweiterung
STRG+F5      Sortieren nach Datum/Zeit
STRG+F6      Sortieren nach Größe
STRG+F7      Unsortiert
STRG+F8      Verzeichnisbaum
STRG+Umsch+F8
             Separater Verzeichnisbaum (durchschalten: einer, zwei, keiner)
STRG+F9      Drucke Datei via verknüpfte Anwendung
STRG+F10     Alle Dateien anzeigen
STRG+F11     Nur Programme anzeigen
STRG+F12     Benutzerdefiniert Dateien anzeigen
TAB          zwischen linken und rechtem Verzeichnisfenster wechseln
UMSCH+TAB    zwischen Verzeichnisfenster und separatem Verzeichnisbaum wechseln
Buchstabe    Eingabe in die Kommandozeile, Cursor wechselt in Kommandozeile
EINFÜGEN     Datei oder Verzeichnis markieren
LEERTASTE    Datei oder Verzeichnis markieren (wie EINF). Falls die LEERTASTE bei
             einem nicht markierten Verzeichnis unter dem Cursor gedrückt wird,
             so wird der Inhalt des Verzeichnisses aufsummiert und seine Größe
             anstelle von <DIR> angezeigt (Modus Ansicht - Lang).  Dies kann via
             'Konfigurieren' - 'Einstellungen' - 'Operation' - 'Markieren mit der
             Leertaste' abgeschaltet werden.
Enter        Verzeichnis wechseln/Programm starten/ Assoziiertes Programm aufrufen
             bzw. Kommandozeile ausführen, falls etwas drin steht. Sofern das 
             Quellverzeichnis den Inhalt einer gepackte Datei anzeigt, werden
             weitere Informationen zur Datei unter dem Cursor angezeigt.
UMSCH+ENTER  1. Kommandozeile bzw. Programm unter Cursor via Windows-Shell aufrufen
                und am Schluss Fenster nicht automatisch schließen (nützlich
                bei Umleitungen > oder <). Geht nur wenn NOCLOSE.EXE im Totalcmd-
                Verzeichnis steht!
             2. Bei ZIP-Dateien: Alternative Wahl aufrufen (wie Verz. behandeln
                <-> externe Packershell wie Winzip)
             3. Innerhalb einer Archivdatei: Entpacke Datei unter Cursor in
                temporäres Verzeichnis und öffne sie als Archivdatei (Bearbeiten
                von zip in zip).
             4. In der Liste der zuletzt benutzten Verzeichnisse (Verlauf, Strg+D)
                öffne das angewählte Verzeichnis in einem neuen Tab.
ALT+UMSCH+ENTER
             Inhalte aller Verzeichnisse aufsummieren. Die Größen werden in der
             Ansicht "lang" anstelle von <DIR> angezeigt.
ALT+ENTER    Zeige Eigenschaftendialog.
STRG+A       Alles markieren
STRG+B       Zeige Inhalt des aktuellen Verzeichnisses inkl. aller Unterverzeichnisse
STRG+Umsch+B Zeige alle markierten Dateien, und alle in allen markierten Unterverzeich-
             nissen (markierte Zweigansicht)
STRG+C       In Zwischenablage kopieren
STRG+D       Verzeichnismenü öffnen
STRG+F       FTP-Verbindung aufnehmen
STRG+UMSCH+F FTP-Verbindung, Portverbindung und gewisse Dateisystem-Plugin-Verbindungen trennen
STRG+I       Ins Zielfenster wechseln
STRG+L       Speicherbedarf (der markierten Dateien) ermitteln
STRG+M       Mehrfach-Umbenenn-Tool
STRG+Umsch+M FTP-Transfermodus ändern
STRG+N       Neue FTP-Verbindung (Eingabe der URL oder Host-Adresse)
STRG+P       Aktuellen Pfad in Kommandozeile kopieren
STRG+Q       Schnellansichtsfenster anstelle eines Dateifensters anzeigen
STRG+Umsch+Q Separates Schnellansichtsfenster (wie Lister, aber wird beim Bewegen in den Dateilisten jeweils angepasst)
STRG+R       Quellverzeichnis neu einlesen
STRG+S       Schnellfilterdialog öffnen und Filter aktivieren
STRG+Umsch+S Schnellfilterdialog öffnen und zuletzt aktiven Filter aktivieren
STRG+T       Neuen Ordner-Tab erzeugen und aktivieren
STRG+Umsch+T Neuen Ordner-Tab erzeugen, aber nicht aktivieren
STRG+U       Verzeichnisse vertauschen
STRG+Umsch+U Verzeichnisse und Ordner-Tabs vertauschen
STRG+V       Dateien aus Zwischenablage einfügen
STRG+W       Aktuellen Tab schließen
STRG+Umsch+W Alle offenen Tabs schließen
STRG+X       In Zwischenablage verschieben
STRG+Z       Kommentar einer Datei bearbeiten
STRG+Oben    Datei unter Cursor in neuem Tab öffnen
STRG+Umsch+Oben
             Datei unter Cursor in neuem Tab im Zielfenster öffnen
STRG+Tab     Zum nächsten Ordner-Tab springen
STRG+Umsch+Tab
             Zum vorherigen Ordner-Tab springen

ALTGR+Buchstabe(n) oder
STRG+ALT+Buchstabe(n)
             Positioniere Cursor auf Datei oder Verzeichnis, die mit diesen
             Buchstaben beginnen. Kann in ALT+Buchstabe(n) abgeändert werden.
             (Unterstützt Tastenkombinationen Strg+C, Strg+X, Strg+V und
              Strg+A; Strg+S für Suchfilter ein/aus)

Kommandozeile:

ENTER            Die Kommandozeile wird ausgeführt, falls sie mindestens ein
                 Zeichen enthält (sonst wird das im Quellverzeichnis momentan
                 markierte Programm gestartet). Ist der Befehl entweder cd, md
                 oder rd, wird er intern direkt ausgeführt. Ist er ein anderer
                 DOS-Befehl, wird DOS gestartet. Sonst wird versucht, ein
                 Programm des angegebenen Namens zu starten.
UMSCH-ENTER      Wie ENTER, aber Start via Windows-Shell. Fenster wird
                 nach Programmende nicht automatisch geschlossen. Geht nur wenn
                 NOCLOSE.EXE im Totalcmd-Verzeichnis steht!
STRG-ENTER       Die aktuell markierte Datei im Quellverzeichnis wird hinten 
                 an die Kommandozeile angehaengt.
STRG-UMSCH-ENTER Die aktuell markierte Datei inklusive Suchpfad wird an die 
                 Kommandozeile angehaengt.
STRG-CURSORDOWN
oder ALT+F8      Eine Liste mit den zuletzt eingegebenen Befehlen (History-Liste)
                 wird aufgeklappt.
TAB              Wechselt den Cursor zurück ins Quellverzeichnis.
Cursortasten     Cursor links und Cursor rechts bewegen den Cursor innerhalb der
                 Kommandozeile, auch wenn er vorher im Quellverzeichnis stand.
                 Cursor auf und Cursor ab bewegen den Cursorbalken im Quell-
                 verzeichnis, auch wenn er vorher in der Eingabezeile stand.
ESC              Löscht Kommandozeile, Cursor zurück ins Dateifenster
STRG+A           Ganze Kommandozeile markieren
STRG+Y           Löscht Kommandozeile, Cursor bleibt in Kommandozeile
STRG+C           Markierten Text in Zwischenablage kopieren
STRG+X           Markierten Text in Zwischenablage verschieben
STRG+V           Text aus Zwischenablage einfügen
STRG+E           Vorherigen Befehl in Kommandozeile kopieren
STRG+K           Bis Zeilenende löschen
STRG+W           Wort links vom Cursor löschen
STRG+T           Wort rechts vom Cursor löschen
